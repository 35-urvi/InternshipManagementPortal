# University Internship Management Portal

## 🚀 Project Overview
A full-stack web application that connects students, companies, and administrators for internship management.

## 🛠 Tech Stack
- Angular
- ASP.NET Core Web API
- SQL Server
- Entity Framework Core
- JWT Authentication

## 👨‍💻 Features
- Role-based login (Student, Company, Admin)
- Internship CRUD operations
- Search & filter internships
- Application tracking

## ⚡ How to Run
1. Run backend (.NET)
2. Run frontend (Angular)
3. Connect database
